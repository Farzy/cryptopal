# Cryptopals

Try to solve https://cryptopals.com/ challenges and learn some cryptography on my own.

# Building

```
cargo build
```

# Running

```
cargo run
```

# Debugging

Prefix the binary with `RUST_LOG=cryptopals=debug` in order to
display debug messages.
